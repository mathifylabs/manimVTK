## Coordinate Systems

### Axes

```python
Axes(
    x_range=[-7, 7, 1],
    y_range=[-5, 5, 1],
    **kwargs
)
```

2D coordinate axes.

### NumberPlane

```python
NumberPlane(
    x_range=[-10, 10, 1],
    y_range=[-10, 10, 1],
    **kwargs
)
```

Grid plane with axes.

### ThreeDAxes

```python
ThreeDAxes(
    x_range=[-6, 6, 1],
    y_range=[-5, 5, 1],
    z_range=[-4, 4, 1]
)
```

3D coordinate system.

## Graphing

### Plotting Functions

```python
axes = Axes(x_range=[-3, 3], y_range=[-2, 2])
graph = axes.plot(lambda x: x**2, color=BLUE)
```

### Labels

```python
axes = Axes()
labels = axes.get_axis_labels(x_label="x", y_label="f(x)")
```
